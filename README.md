## 正しいデスク姿勢のポイント(自分用)
### 1. モニター位置
- 画面の上端が目の高さと同じくらいになるように調整。
- モニターまでの距離は腕を伸ばして指先が届く程度（約50〜70cm）。

### 2. 椅子と座り方
- 背もたれに軽くもたれ、骨盤を立てる（背筋を伸ばす）。
- 腰の後ろにクッションや丸めたタオルを入れると良い。
- 膝は股関節より少し低め、足裏は床にしっかり接地。

### 3. 腕と手の位置
- 肘は90度前後で、肩をすくめないように。
- マウスやキーボードは体の近くに置き、腕を前に伸ばしすぎない。

### 4. 休憩・ストレッチ（1時間に1回が理想）
1. 肩甲骨リセット体操（おすすめ）
2. 背筋を伸ばして座る／立つ
3. 両肩を「すくめる → 下ろす」を5回
4. 肩甲骨を「寄せる → 開く」を5回
5. 両腕を後ろで軽く組み、胸を開くように10秒キープ
